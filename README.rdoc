== README

This a sample socail network site that allow you to sign up and connect with other user.

You can make new post or edit or delete post. And the change will show up in your friend's news feed.

To search for new friend simply type in the name of the friend and hit enter.

Then you can send friend ship request to that particular friend.

Likewise if some one send a friendship request to you, you can approve or deny their request.
